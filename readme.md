# README

## 1. Title: Sorting Real Numbers

## 2. Description
The *Sorting Real Numbers* system collects a set of real number and sort them in asceding order.

## 3. Features
- Collect a set of real numbers
- Sort them in ascending order
- Displey them

## 4. Requirements
A C compiler language compatible with ANSI C.

### 4.1 Dependencies
- `<stdio.h>` standard input/output library

## 5. Instalation
You can clone the project from github
- **Source:** `git clone https://github.com/AmauriDev/EX02C13.git`
- **CodeBlock IDE:** [Download](https://www.codeblocks.org/downloads/)

Open `02-exercise.cbp` containts all information to work on `CodeBlock`

## 6. Usage
**Windows platform**
Inside the project forder navigate to `build/` directory.
Open a CMD windows and run
`02-exercise`

## 7. Project structure
- `project-root/`
  - `build/` - compiled executable
  - `docs/`  - documentation files
  - `src/`   - source code

## 8. Modules
- `get_set_of_real_numbers.c` - subsystem collects a set of real numbers from the user
- `sort_real_numbers.c`       - subsystem sort them in ascending order
- `display_numbes.c`          - display them to the user
